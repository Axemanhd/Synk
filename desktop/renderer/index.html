Setup Guide


        

          
1. Create a Discord Application

          
    
            
  1. Go to the Discord Developer Portal and click New Application.
    2. 
            
  2. Give your application a name (e.g., "Synk") and click Create.
    3. 
          

        


        

          
2. Get Your Client ID

          
    
            
  1. In your application page, go to OAuth2 > General.
    2. 
            
  2. Copy the Client ID value.
    3. 
            
  3. Paste it into the Settings page under DISCORD_CLIENT_ID.
    4. 
          

        


        

          
3. Create a Bot User

          
    
            
  1. Go to Bot in the left sidebar.
    2. 
            
  2. Click Add Bot > Yes, do it!
    3. 
            
  3. Under the Privileged Gateway Intents section, enable:
              
      
                
    • MESSAGE CONTENT INTENT (required for commands)
      • 
                
    • SERVER MEMBERS INTENT (recommended)
      • 
              
    
            
    4. 
          

        


        

          
4. Get Your Bot Token

          
    
            
  1. Still under Bot, click Reset Token (or copy the existing one).
    2. 
            
  2. Copy the token and paste it into the Settings page under DISCORD_TOKEN.
    3. 
            
  3. Warning: Never share your bot token. If your token is leaked, reset it immediately on the Developer Portal.
    4. 
          

        


        

          
5. Invite the Bot to Your Server

          
    
            
  1. Go to OAuth2 > URL Generator.
    2. 
            
  2. Select the following scopes:
              
      
                
    • bot
      • 
                
    • applications.commands
      • 
              
    
            
    3. 
            
  3. Select these bot permissions:
              
      
                
    • Send Messages
      • 
                
    • Manage Messages
      • 
                
    • Embed Links
      • 
                
    • Read Message History
      • 
                
    • Connect (voice)
      • 
                
    • Speak (voice)
      • 
                
    • Use Voice Activity
      • 
              
    
            
    4. 
            
  4. Copy the generated URL and open it in your browser.
    5. 
            
  5. Select your server and click Authorize.
    6. 
          

        


        

          
6. Install FFmpeg (Required for Audio)

          
    
            
  1. If you haven't already, install FFmpeg from ffmpeg.org.
    2. 
            
  2. Or install via WinGet by running in a terminal: winget install Gyan.FFmpeg
    3. 
            
  3. Make sure FFmpeg is available in your system PATH.
    4. 
          

        


        

          
7. Run the Bot

          
    
            
  1. Make sure you've saved your DISCORD_TOKEN and DISCORD_CLIENT_ID in the Settings page.
    2. 
            
  2. Go to the Dashboard tab and click Run Bot.
    3. 
            
  3. Check the console output to verify the bot connects successfully.
    4. 
            
  4. Use /play in your Discord server to start playing music!
    5. 
          

        


        

          
Available Commands

          

            
CommandDescription

            
/playPlay a song from YouTube or Spotify

            
/skipSkip the current track

            
/stopStop playback and clear the queue

            
/pausePause or resume playback

            
/queueView the current song queue

            
/volumeAdjust the playback volume

            
/leaveDisconnect the bot from voice
